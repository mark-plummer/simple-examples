= Kroki examples with bytefield and kref
:kroki-default-format: svg
:kroki-default-options: inline

include::partial$sample.adoc[]

Here's a plain bytefield diagram:

[sample]
----
[bytefield]
....
(draw-column-headers)
(draw-box "Address" {:span 4})
(draw-box "Size" {:span 2})
(draw-box 0 {:span 2})
(draw-gap "Payload")
(draw-bottom)
....
----

In this diagram, the `Address` box is a link:

[sample]
----
[bytefield,subs="+macros,attributes"]
....
(draw-column-headers)
(wrap-link kref:index.adoc[]
  (draw-box "Address" {:span 4}))
(draw-box "Size" {:span 2})
(draw-box 0 {:span 2})
(draw-gap "Payload")
(draw-bottom)
....
----
